[기획 정리]

------------------------
1. 안드로이드 출시 (Mission)
   : 0 -> 1 특정 카테고리의 안드로이드를 출시한다.
   : 1 -> ∞ 버전 업
   : 레벨에 따라서 시간당 일정량의 돈
   : 버전 업에 드는 돈이 많아져서 결국 머무름
   : 새로운 안드로이드 출시 마다 이벤트 및 CG

   : 어떤 안드로이드를 출시할 지 차후 리스트는 노출되지 않는다.

2. 활동 (휴식, 특강 등) (선불) (Learn)
   : 돈을 내고 일정 시간 기다려서 파라미터 올리기
   : 리스트에 올리기 방식으로 변경
   : 현재 방식으론 서로 다른 것만 1234를 정할 수 있음
   : 새로운 용도의 안드로이드 출시 조건 개방

   : 각종 조건들로 인해 가능한 활동이 늘어난다.
   : 차후 리스트는 노출되지 않는다.

   : 배터리만 소모하고 시간은 들지 않음.
   : 시간적인 밸런싱은 모두 Mission 으로 빼둠.

3. 아이템 (스킬)
   : 효율을 높여주는 아이템들.
   : 패시브도 있고 액티브도 있다.
   : 액티브는 화면으로 빼줘야 하나?

   : Learn 행동의 성공 확률을 높인다.
   : Learn 행동의 선택지가 늘어난다.
   : Learn 행동에 학습 조너스가 붙는다.
   : Mission 행동의 선택지가 늘어난다.
   : Mission 행동의 생산량이 많아진다.
   : 총 전력량이 늘어난다.
   : 전력 소모량이 줄어든다.
   : 버전 업 시 효율이 좋아진다.   

4. 배터리
   : 안드로이드의 제한 시간
   : 실제 시간으로도 줄어들고 활동마다도 시간을 쓴다.
   : 기본적으로 현실 시간 일주일 단위.

   배터리가 다 되면?

   패러미터에 따른 엔딩.
   
   다시 되돌아옴.
   일부 에너지 코어를 얻고 다시 0부터

   되돌아오면 아이템은 유지되어 있음.
   왜?
   그걸 밝혀 내는 게 이 게임의 중심 스토리임.
----------------------------

게임 플로우 시나리오

- 시작 스토리 (배경 설명)
- 안드로이드 출시 기능 열림
- 안드로이드 출시 초기 1 버전 1이 돈을 벌어주기 시작함
- 가만히 있어도 돈이 벌린다는 걸 알려줌

- 학습에 대한 스토리
- 큐에 등록하면 확인 후 이벤트 진행할 수 있음
- 전력이 소모됨
- 1/2/3/4 큐가 흘러가는 걸 봄
- 스트레스가 올라가서 무한정 진행 못함
- 스트레스 내리기 위한 행동엔 돈이 많이 듬

- 돈을 많이 벌면 무한히 능력치 올릴 수 있을 거야
- 안드로이드 버전 업 알려줌

- 돈과 학습을 오가다 보면 이벤트 스토리가 나옴
- 능력치 조건에 따라서 여러 가지 나옴

- 일정 이상 성장하면 아이템 구매할 수 있게 됨
- 효율이 좋아짐
- 자신만의 학습 빌딩이 달라짐 (보고 싶은 이야기가 다르니까?)

- 전력이 다함
- 1차 엔딩

- 아이템을 가진 채로 다시 초기화된 상태로 시작
